# Florist Date Reminder Web App

A lightweight, browser-based tool for customers to save important dates (anniversaries, birthdays, etc.) directly to their phone's native Reminders or Calendar app.

## 🌸 How it Works
This app allows users to:
1. **Select a Date:** Using a simple calendar interface.
2. **Add a Note:** Personalize the reminder (e.g., "Mom's Birthday").
3. **Set an Alert:** Choose how many days in advance they want to be notified.
4. **Direct Link:** Every reminder includes a link back to our florist website to make ordering easy.

## 📱 Installation (No App Store Required)
This is a **Progressive Web App (PWA)**. To "install" it on a smartphone:
* **iOS:** Open the link in Safari, tap the 'Share' icon, and select **'Add to Home Screen'**.
* **Android:** Open the link in Chrome, tap the menu (three dots), and select **'Install'** or **'Add to Home Screen'**.

## 🛠 Technical Details
* **Hosting:** Hosted for free via GitHub Pages.
* **Format:** Uses the `.ics` (iCalendar) standard to communicate with phone hardware.
* **Privacy:** No user data is stored on a server. All reminder generation happens locally in the user's browser.
